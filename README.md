# Barra Hub - Turismo em Barra Bonita

[![Website](https://img.shields.io/badge/Website-Online-brightgreen)](https://KarinaNicoleti.github.io/barra-hub/)

Barra Hub é um site responsivo que apresenta as principais atrações turísticas, restaurantes e hotéis de Barra Bonita, SP. O projeto utiliza **HTML, CSS e JavaScript**, oferecendo uma experiência visual moderna e interativa.

---

## Preview

![Preview do site](assets/preview.png)  
*Exemplo da página inicial com o banner e cards de atrações.*

---

## Funcionalidades

- Página inicial com destaque para os passeios principais  
- Cards informativos de passeios, restaurantes e hotéis  
- Menu responsivo para dispositivos móveis  
- Botão "Saiba mais" em cada card para detalhes  
- Seção de contato no rodapé  

---

## Tecnologias Utilizadas

- **HTML5**  
- **CSS3** (Flexbox e Grid)  
- **JavaScript** (para menu mobile e interações)  
- **Google Fonts** (Poppins)
